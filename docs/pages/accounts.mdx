# Account Implementations

There are various implementations for ERC-6900 accounts including:

## Alchemy Modular Account

A maximally modular, efficient, upgradable smart contract account that is compatible with ERC-4337 v0.7 and ERC-6900 v0.8.

Built by [Alchemy](https://www.alchemy.com/).

<GithubLink href="https://github.com/alchemyplatform/modular-account" />

## Circle Smart Accounts

Leverages passkey signing for seamless Web3 integration and supports advanced use cases through modules. Compatible with ERC-4337 v0.7 and ERC-6900 v0.8.

Built by [Circle](https://www.circle.com/).

<GithubLink href="https://github.com/circlefin/buidl-wallet-contracts/tree/master/src/msca/6900/v0.8" />
